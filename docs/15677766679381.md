# 第 8 章接口


### 3.2 适配器模式

1. 继承方式实现
2. 组合方式实现

组合关系比继承关系更有利于系统的维护和扩展，而且组合关系能够将多个源接口转换为一个目标接口。

继承关系只能把一个源接口转化为一个目标接口，优先考虑使用组合关系来实现适配器。

## 4 小结

接口中的成员变量和方法必须都是 public 类型。

接口和抽象不同点:
-  抽象类中可以定义实例变量，接口中的变量都是静态的
